Frequently Asked Questions

INFORMASI UMUM

Apakah data saya tetap aman saat menggunakan keyboard Shopkeepr?

 

Shopkeepr memastikan data kamu tetap aman saat menggunakan keyboard Shopkeepr, karena Shopkeepr tidak menyimpan informasi atau data apapun yang kamu ketik menggunakan keyboard Shopkeepr.

Keyboard Shopkeepr berfungsi untuk mengetik pesan karakter melalui smartphone. Dalam proses mengetik, Shopkeepr tidak menyimpan data apapun yang telah kamu ketik menggunakan Shopkeepr.

Setelah install aplikasi Shopkeepr, kamu akan mendapat popup pesan di smartphone kamu yang berisi peringatan penggunaan keyboard. Peringatan ini adalah default pesan dari sistem untuk langkah awal penggunaan aplikasi pihak ketiga. Peringatan ini merupakan bagian dari sistem operasi Android yang muncul untuk semua jenis instalasi apliksi pihak ketiga, termasuk keyboard Shopkeepr.

Tetapi seperti yang sudah dijelaskan sebelumnya, kami memastikan bahwa Shopkeepr sama sekali tidak menyimpan informasi atau data yang kamu ketik menggunakan keyboard Shopkeepr.

KEYBOARD SHOPKEEPR

Bagaimana cara mengaktifkan keyboard Shopkeepr?

  1. Masuk ke aplikasi Shopkeepr.
  2. Tekan menu bar di pojok kanan atas.
  3. Pilih “Aktivasi Keyboard”. Kamu akan diarahkan ke langkah mengaktifkan keyboard Shopkeper.
  4. Pilih menu “AKTIFKAN DI PENGATURAN”, lalu aktifkan “Shopkeepr” . Tekan “Ok” untuk menyetujui ketentuannya.
  5. Pada menu “PILIH MASUKAN BAHASA”, pilih “Shopkeepr”.

Keyboard Shopkeepr sudah aktif dan bisa digunakan sebagai asisten kamu untuk mengurus toko online kamu.

Shopkeepr memastikan data kamu tetap aman saat menggunakan keyboard Shopkeepr, karena Shopkeepr tidak menyimpan informasi atau data apapun yang kamu ketik menggunakan keyboard Shopkeepr. Kamu bisa membaca lebih lengkap mengenai keyboard Shopkeepr di bagian Informasi Umum.

Simak juga video tutorial cara mengaktifkan keyboard Shopkeepr

 

Bagaimana cara menonaktifkan keyboard Shopkeepr?

  1. Masuk ke aplikasi Shopkeepr.
  2. Tekan menu bar di pojok kanan atas.
  3. Pilih “Aktivasi Keyboard”. Kamu akan diarahkan ke langkah sama seperti saat mengaktifkan keyboard Shopkeper.
  4. Pilih menu “AKTIFKAN DI PENGATURAN”, lalu nonaktifkan “Shopkeepr” .
  5. Pada menu “PILIH MASUKAN BAHASA”, lalu nonaktifkan “Shopkeepr”.

 

Kamu juga bisa menonaktifkan keyboard Shopkeepr dengan cara singkat berikut:

  1. Masuk ke chat pelanggan kamu.
  2. Tekan lama pada spaci di keyboard Shopkeepr.
  3. Lalu nonaktifkan keyboard Shopkeepr.

Sekarang, keyboard Shopkeepr berhasil dinonaktifkan.

QUICK REPLY

Bagaimana cara menambahkan pesan balas cepat (Quick Reply)?

  1. Masuk ke chat pelanggan kamu.
  2. Buka keyboard Shopkeepr.
  3. Pilih ikon Quick Reply di toolbar.
  4. Klik tombol untuk menambahkan pesan balas cepat baru.
  5. Masukkan shortcut dan deskripsi di kolom yang tersedia.
  6. Klik “SIMPAN” untuk menyimpan Quick Reply.

Sekarang kamu sudah bisa menggunakan Quick Reply yang kamu buat untuk membalas konsumen dengan lebih cepat.

Simak juga video tutorial cara menambah Quick Reply

 

Bagaimana cara mengubah pesan balas cepat?

  1. Masuk ke chat pelanggan kamu.
  2. Buka keyboard Shopkeepr.
  3. Pilih ikon Quick Reply di toolbar.
  4. Klik ikon di sebelah kanan Quick Reply yang ingin kamu ubah.
  5. Pilih “Ubah”.
  6. Masukkan Shortcut dan Deskripsi di kolom yang tersedia.
  7. Klik “SIMPAN”.

 

Bagaimana cara menghapus pesan balas cepat?

  1. Masuk chat pelanggan kamu.
  2. Buka ke keyboard Shopkeepr.
  3. Pilih ikon Quick Reply di toolbar.
  4. Klik ikon di sebelah kanan Quick Reply yang ingin kamu hapus.
  5. Pilih “Hapus”.
  6. Klik “Ok” pada popup pesan konfirmasi.

 

Bagaimana cara menggunakan Quick Reply untuk menjawab pertanyaan pelanggan dengan cepat?

  1. Masuk ke keyboard Shopkeepr.
  2. Ketik karakter “\” (yang ada di kiri spasi) diikuti shortcut yang ingin dikirim.
  3. Klik shortcut yang muncul di toolbar.
  4. Quick Reply akan muncul otomatis di kolom chat dan siap dikirim untuk membalas pertanyaan pelanggan.

 

Apakah yang dimaksud dengan shortcut?

Shortcut adalah kata pintasan untuk mempermudah kamu mencari Quick Reply di keyboard Shopkeepr.

CEK ONGKIR

Bagaimana cara cek ongkir?

  1. Masuk ke chat pelanggan kamu.
  2. Buka keyboard Shopkeepr.
  3. Masukkan berat barang, kota asal dan kota tujuan di kolom yang tersedia.
  4. Klik “Cek Ongkir”.
  5. Hasil ongkir dari berbagai jasa pengiriman akan muncul.

 

Bagaimana cara mengirim hasil ongkir ke pelanggan?

  1. Masuk ke chat pelanggan kamu.
  2. Buka keyboard Shopkeepr.
  3. Masukkan berat, kota asal dan kota tujuan di kolom yang tersedia.
  4. Klik “Cek Ongkir”.
  5. Hasil ongkir dari berbagai jasa pengiriman akan muncul.
  6. Tandai kotak di kiri logo kurir yang kamu inginkan untuk dikirim ke pelanggan .
  7. Klik “Salin”.
  8. Hasil ongkir akan tersalin otomatis di kolom chat dan siap dikirim ke pelanggan.

KALKULATOR

Bagaimana cara menggunakan kalkulator?

  1. Masuk ke keyboard Shopkeepr.
  2. Pilih ikon Kalkulator di toolbar.
  3. Kamu dapat melakukan perhitungan dengan mudah.

 

Apakah saya bisa menyalin hasil perhitungan saya di kalkulator melalui keyboard Shopkeepr?

Ya. Kamu bisa menyalin perhitungan kamu dengan mudah, dengan cara:

  1. Klik ikon “Salin” di bagian kanan hasil perhitungan.
  2. Tempelkan hasil perhitungan di chat room. Kamudian bisa langsung dikirim ke pelanggan kamu.
